# 货车宝司机夏季换胎无忧 H5

## 当前版本

这是一个无需安装依赖即可打开的移动端活动页面原型。

已实现：

- 主推单品：玲珑轮胎 四线花纹 12R22.5；
- 活动统一报价：多城司机团购折扣后999元/条，需旧胎回收；
- 货车宝平台支持50元抵扣券，线上支付自动抵扣；
- 顶部使用货车宝 H logo，活动主题色统一为 logo 蓝色 `#3366ff`；
- 顶部展示多城司机正在团引导，配合圆角头像、风景图、卡通图轮播；
- 包安装、正品三包；
- 顶部只突出活动和优惠，产品规格与优势集中放在“确认轮胎和数量”模块；
- 司机确认轮胎数量（最多10条），并填写联系电话、预计到店周期、预约门店；
- 页面展示玲珑四线花纹轮胎产品图；
- 到店周期为近3天、近7天、1个月内、1个月及以上；
- 首批西安、义乌、杭州钱塘、东莞、徐州、山东潍坊试点门店；
- 门店下拉增加“其他城市正在陆续开放，敬请期待！”选项，选中后不可提交支付；
- 预约门店模块精简为门店选择、地址展示和一次性服务优势标签：专业换胎、供应稳定、本地300+司机推荐；
- 系统自动按999元/条计算商品金额，并扣减50元平台抵扣券；
- 支付前回显完整订单，支持模拟在线支付。

## 适老化阅读设计

- 删除英文、装饰图形和重复宣传信息；
- 基础正文采用17—20px，关键标题27—54px；
- 商品价格采用38px以上高对比红色；
- 主要按钮高度不低于58px；
- 页面只保留轮位、数量、三个预约字段和支付金额；
- 采用白底黑字、黄色选中状态，减少低对比度内容；
- 不要求司机选择具体日期和时间段，降低填写门槛。

## 本地打开

直接双击 `index.html` 即可查看。

也可以在当前目录运行：

```bash
python3 -m http.server 8080
```

然后访问：

```text
http://127.0.0.1:8080
```
