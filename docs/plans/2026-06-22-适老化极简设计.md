# 夏季换胎无忧H5适老化极简设计

## 目标用户

以40岁以上货车司机为主要用户，兼顾老花眼、户外强光、操作时间短和对复杂电商页面不熟悉等情况。

## 核心原则

1. 一屏只解决一个问题；
2. 关键价格必须一眼可见；
3. 正文不使用小于16px的字号；
4. 主要点击区域不低于56px；
5. 不使用英文、装饰性信息和重复承诺；
6. 不依赖颜色单独表达选中状态，同时提供粗边框和对勾；
7. 使用司机容易理解的自然语言，例如“近3天、包安装、999元/条、需旧胎回收”。

## 信息顺序

1. 活动是什么；
2. 轮胎规格和服务承诺；
3. 每单优惠金额；
4. 选择轮位和数量；
5. 选择安装门店；
6. 选择到店时间；
7. 填写司机信息；
8. 确认订单金额并支付。

## 视觉规范

- 背景：白色；
- 正文：近黑色；
- 选中及行动色：黄色；
- 价格及优惠：红色；
- 正向库存状态：绿色；
- 正文：17—20px；
- 小标题：27px；
- 活动规格：46—54px；
- 商品价格：38px；
- 实付金额：44px；
- 按钮高度：58—64px。

## 验收标准

- 390px宽手机屏幕下无需放大即可阅读；
- 首屏能够看清规格、包安装、三包、优惠和首个价格；
- 所有核心按钮单手容易点击；
- 商品、数量、门店和时间选择后金额正确变化；
- 支付前清晰展示商品、门店、时间、司机、优惠和实付金额。
